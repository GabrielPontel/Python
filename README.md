# Introdução a Python

Repositório contendo resoluções de exercícios desenvolvidas ao longo de cursos em Python, realizados por Gabriel Pontel de Mori no ano de 2025.

## _Conteúdo Abordado_:

- Estruturas Condicionais (if, else e elif);
- Estruturas de Repetição (for e while);
- Estruturas de dados (Lista, Dicionário e Tupla);
- Bibliotecas;
- Funções;
  - built-in functions (funções embutidas);
  - Funções próprias;  
  - Lambda Functions (funções anônimas).
